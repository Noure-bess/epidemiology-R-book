6.1 Ordinal scales

Different from the ratio (continuous data) and the nominal (discrete data) scales, the ordinal scales are structured as rank-ordered numeric classes in a finite number of classes. There are two kinds of ordinal scales: quantitative and qualitative (Chiang and Bock 2021). There reasons for using ordinal scales include convenience and speed of rating (Madden et al. 2017).

6.1.1 Quantitative ordinal scale

In the quantitative scale, each score represents a defined interval of the percentage scale. The most well-known quantitative ordinal scale is the Horsfall-Barratt (HB) scale, developed in the early 1940s when the science of plant pathology was in need of more quantitative approaches (Hebert 1982). The HB scale splits the percent scale into 12 consecutive logarithmic-based intervals of severity from 0 to 100% . The consecutive intervals increase in size from 0 to 50% and decline in size from 50 to 100%.

Warning

The divisions of the H-B scale were based on two assumptions. First, on the logarithmic relationship between the intensity of a stimulus and sensation. Second, that the rater tends to focus on objects that are of smaller size when observing objects of two colors (Madden et al. 2017). The topic is point of controversy in the literature, as there is little experimental evidence of these assumptions. To date, evidence shows a linear relationship between visually estimated and actual severity (Nutter and Esker 2006).

To overcome the problems associated with the H-B scale (Bock et al. 2013), a new scale was recently proposed, following a linear relationship with percentage area diseased at severities >10% (class 6 on the scale) (Liu et al. 2019; Chiang et al. 2014)

6.1.2 Qualitative ordinal scale

In the qualitative ordinal scales, each rating provides a description of the symptoms. An example is the ordinal 0-3 scale for rating eyespot of wheat developed by (Scott and Hollins 1974).

Ordinal scale for rating eyespot of wheat (Scott and Hollins 1974)
Class Description
0 uninfected
1 slight eyespot (or or more small lesion occupying in total less than half of the circumference of the stem)
2 moderate eyespot (one or more lesions occupying at least half the circumference of the stem)
3 severe eyespot (stem completely girdled by lesions; tissue softened so that lodging would really occur)

6.2 Disease severity index (DSI)

Usually, when quantitative or qualitative ordinal scales are used, the scores are transformed into an index on a percentage basis, such as the disease severity index (DSI). The DSI is a single number that summarizes a large amount of information on disease severity (Chester 1950). The formula for a DSI (%) can be written as follows:

\(DSI = \frac{∑(class \ frequency \ ✕ \ score \ of \ rating \ class)} {total \ number \ of \ observations \ ✕ \ maximal \ disease \ index} ✕ 100\)

Let’s create a function named DSI() to automate the calculation of the disease severity index (DSI) in a series of units (e.g. leaves) for which one of the scores of the ordinal scale has been assigned during the evaluations.

DSI <- function(unit, score, max) {
df <- data.frame(unit, score)
tab <- df |> 
janitor::tabyl(score)
tab2 <- tab |> 
  dplyr::mutate(weight = score * n) 
(sum(tab2$weight))/(nrow(df)*max)*100
}

The function requires three arguments:

  • unit = the vector of the number of each unit

  • score = the vector of the scores for each unit

  • max = the maximum value of the ordinal scale.

Let’s use with a toy data set composed of 12 units where each received an ordinal score. The vectors were arranged as a dataframe named scores.

unit <- c(1:12)
score <- c(2,3,1,1,3,4,5,0,2,5,2,1)
scores <- data.frame(unit, score)
